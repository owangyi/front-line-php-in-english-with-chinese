# 前言
作者：Roman Pronskiy

PHP 改变了世界。如今，全球有超过 500 万开发者使用这种语言来创建各种应用程序。同时世界也在改变 PHP。PHP 之父 Rasmus Lerdorf 曾指出，"一个好的解决方案应该借鉴现有技术"。确实如此，PHP 一直在从其他生态系统中汲取最佳创意并加以改进。

自 PHP 7.0 发布以来已经过去了 5 年。在这段时间里，随着社区的成熟和对新功能的需求，语言团队一直在倾听社区的声音并实现这些功能。这就是 PHP 引入更严格的类型系统、更简洁的语法、静态分析器等的原因。

"生存下来的不是最强壮的物种，也不是最聪明的物种，而是最能适应变化的物种。"——Leon C. Megginson 对达尔文思想的诠释，这一见解年复一年地指导着 PHP 的发展。

当我第一次看到 PHP 时，我还在上学，当时我完全没想到我会继续专业地使用它。那时，在 2004 年，互联网的主要问题是搜索信息。如今，由于有如此多的信息唾手可得，问题不再是搜索信息，而是过滤和识别哪些是有用的。用一个老套的比喻来说，就像在干草堆里找针——或者反过来？无论如何，我们如何在这些混杂着广告和通知的无尽信息流中挖掘出真正有价值的东西？

不可能对所有领域都给出全面的答案，但对于 PHP，答案就在这里 —— Front Line PHP。

这本书提供了关于 PHP 今天和近期发展的一切核心知识。这不是文档的转述，而是以简短清晰的表述呈现的深思熟虑且有意义的开发经验。它提供了对语言最新功能的新视角，以及对一些已经熟悉的功能的新视角。

用 PHP 编写高质量的代码同时变得既更容易又更复杂。一方面，开发者可以使用许多新功能和工具。另一方面，你需要优雅地运用它们，以便漂亮而轻松地解决实际问题。

这就是这本书的真正意义所在。用简单的语言解释复杂的事物。如何应用它们。原则和实践。它是一根钓鱼竿，在满是鱼的海洋中。

使用 PHP 从未有过更好的时机。而这本书是学习它的最佳方式之一。
