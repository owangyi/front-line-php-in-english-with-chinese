# 序言

当我开始编写 PHP 时，很少有高质量的学习资源。许多在线资源要么已经过时，要么专注于初学者。所以大约四年前，我决定以高质量的方式为 PHP 社区做出贡献。我开了一个博客，现在已经相当受欢迎，写作从未停止。

但博客的缺点是，每篇文章都是独立的小岛。你不能期望访客读过你之前的所有文章。很难从头到尾讲述一个故事，很难在博客上有一个连续的叙述。

与此同时，我很幸运能够在一家重视我之前所有作品的公司工作，那里的每个同事都对他们的工作充满热情。我们一起决定做一些超越个人文章或想法的事情，一些讲述编写现代 PHP 的故事，一些最终会成为这本书的东西。

这本书将成为所有 PHP 开发者的重要垫脚石；无论你是初学者还是专家：你都能学到很多东西。这本书是一个统一的故事，它连接了我们多年来一直在努力的各个点。这是我写作时的个人目标，我相信我们已经做到了这一点。

祝您阅读愉快，让我们开始吧！
